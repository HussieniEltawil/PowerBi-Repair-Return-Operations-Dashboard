# Contributing

Thank you for your interest in this project.

This repository is maintained primarily as a professional portfolio project
demonstrating Power BI, DAX, Power Query, data modeling, and operational
analytics skills.

External code or dashboard contributions are not currently being accepted.

You may still:

- Open an issue to report a broken link
- Suggest a documentation correction
- Report a potential privacy concern
- Share constructive feedback

Please do not upload or request access to any original confidential dataset.
