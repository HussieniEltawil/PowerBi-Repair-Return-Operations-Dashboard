# Security and Data Privacy

This repository contains an anonymized portfolio version of an operational
Power BI project.

No confidential supplier, manufacturer, customer, serial number, part number,
work order, logistics, or personally identifiable information is intended to
be included.

## Reporting a Privacy Concern

If you identify information that may require further anonymization, please do
not create a public issue containing the sensitive information.

Contact the repository owner privately through the contact information
available on the GitHub profile.

Please include:

- The affected file
- The relevant page, table, or field
- A short explanation of the concern

The repository owner will review and, when appropriate, remove or further
anonymize the information.
